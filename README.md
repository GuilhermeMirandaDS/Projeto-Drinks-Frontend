# Catálogo de Drinks

## Descrição

Este projeto é um catálogo interativo de drinks desenvolvido com Vue.js e TypeScript. Ele permite aos usuários explorar uma vasta coleção de receitas de drinks, utilizando a API pública TheCocktailDB. A aplicação oferece funcionalidades de busca robustas e um sistema de favoritos para uma experiência personalizada.

## Funcionalidades

*   **Busca por Nome:** Encontre drinks específicos digitando seus nomes.
*   **Busca por Categoria:** Filtre drinks por diferentes categorias (ex: coquetéis, bebidas quentes, etc.).
*   **Busca por Primeira Letra:** Descubra drinks começando com uma letra específica.
*   **Favoritos:** Salve seus drinks preferidos para acesso rápido, utilizando Pinia para gerenciamento de estado.
*   **Página de Detalhes do Drink:** Visualize informações detalhadas sobre cada drink, incluindo ingredientes, instruções e imagens.

## Tecnologias Utilizadas

*   **Vue.js:** Framework progressivo para construção de interfaces de usuário.
*   **TypeScript:** Superset tipado de JavaScript que melhora a robustez e manutenibilidade do código.
*   **Pinia:** Gerenciador de estado intuitivo e leve para aplicações Vue.
*   **TheCocktailDB API:** Fonte de dados para todas as informações de drinks e receitas.

## Deploy

Atualmente o deploy da aplicação está no Netlify para visualização: `https://drinks-catalog-gms.netlify.app/`

## Uso

Após iniciar a aplicação, você pode:

*   Navegar pela página inicial para ver drinks em destaque.
*   Usar a barra de busca para encontrar drinks por nome, categoria ou primeira letra.
*   Clicar em um drink para ver seus detalhes completos.
*   Adicionar drinks à sua lista de favoritos clicando no ícone de coração.

## Melhorias Futuras

Este projeto está em constante evolução. Novas funcionalidades e otimizações são adicionadas semanalmente para aprimorar a experiência do usuário e a performance da aplicação.

# Cocktail Catalog

## Description

This project is an interactive cocktail catalog developed with Vue.js and TypeScript. It allows users to explore a vast collection of drink recipes using the public TheCocktailDB API. The application offers robust search functionalities and a favorites system for a personalized experience.

## Features

*   **Search by Name:** Find specific drinks by typing their names.
*   **Search by Category:** Filter drinks by different categories (e.g., cocktails, hot drinks, etc.).
*   **Search by First Letter:** Discover drinks starting with a specific letter.
*   **Favorites:** Save your favorite drinks for quick access, using Pinia for state management.
*   **Drink Detail Page:** View detailed information about each drink, including ingredients, instructions, and images.

## Technologies Used

*   **Vue.js:** Progressive framework for building user interfaces.
*   **TypeScript:** Typed superset of JavaScript that improves code robustness and maintainability.
*   **Pinia:** Intuitive and lightweight state management library for Vue applications.
*   **TheCocktailDB API:** Data source for all drink information and recipes.

## Deploy

The application deploy is on Netlify open to use: `https://drinks-catalog-gms.netlify.app/`

## Usage

After starting the application, you can:

*   Browse the homepage to see featured drinks.
*   Use the search bar to find drinks by name, category, or first letter.
*   Click on a drink to view its full details.
*   Add drinks to your favorites list by clicking the heart icon.
